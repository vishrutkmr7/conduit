# Product

## Register

product

## Users

Conduit is for developers and power users who connect remote MCP servers to Apple system surfaces. They use it while setting up automations, checking available tools, and running focused agent tasks from the app, Siri, Shortcuts, widgets, Spotlight, and controls.

## Product Purpose

Conduit turns configured MCP servers into native Apple actions and entities. Success means a user can add a server, inspect its tools, run a task safely, and reuse the same server and tool catalog across system experiences without learning a custom UI vocabulary.

## Brand Personality

Native, calm, trustworthy. The app should feel like an Apple productivity utility: direct labels, restrained surfaces, clear status, and no decorative styling that competes with the task.

## Anti-references

Avoid custom SaaS dashboards, decorative glass panels, neon AI palettes, card grids as the default layout, non-system controls, and visual effects that imitate Apple materials instead of using the real system components.

## Design Principles

- System first: use native SwiftUI controls, semantic colors, system materials, and standard navigation.
- Trust before speed: make authentication, tool risk, errors, and confirmations explicit.
- One catalog everywhere: servers and tools should resolve consistently in app UI, widgets, Siri, Shortcuts, and Spotlight.
- Useful density: prefer scannable lists, sections, labels, and status over marketing-style presentation.
- Accessible by default: every action works with Dynamic Type, VoiceOver, Reduce Motion, and Differentiate Without Color.

## Accessibility & Inclusion

Target full Dynamic Type support, VoiceOver labels for every icon-only affordance, 44x44 minimum touch targets, Reduce Motion alternatives, and non-color status cues. User-facing text should remain concise and localizable.
